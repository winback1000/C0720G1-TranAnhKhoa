Git/GitHub
Các lệnh căn bản

    Source code: Là các file để lập trình
    GitHub: _ Hệ thống quản lý các phiên bản source code, lưu trữ dữ liệu
            _ Dịch vụ dùng để triển khai Git
    Git: Công cụ quản lý các file trên nhiều phiên bản
    Repository: kho lưu trữ các file

Plug in: Tortoise Git
Git clone link RR: Clone Remote Repository về Local Repository
Git add . (Git add file.abc): thêm tất cả các file hoặc một file vào trạng thái chuẩn bị
